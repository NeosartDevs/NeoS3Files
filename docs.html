NeoS3Files

Высокоуровневая асинхронная Python-библиотека для работы с S3-совместимыми хранилищами

Python 3.7+ MIT License v2.0.0 AsyncIO

Полностью асинхронный API

Все операции ввода-вывода неблокирующие, построены на asyncio, aioboto3 и aiofiles.

Автоматическая загрузка частями

Большие файлы автоматически разбиваются на части и загружаются параллельно с контролем конкурентности.

Отслеживание прогресса

Колбэки для мониторинга прогресса загрузки с информацией о завершённых частях и процентах.

Пул соединений

Автоматическое переиспользование HTTP-соединений для максимальной производительности.

Типизированные исключения

Детальная иерархия исключений для всех видов ошибок S3 с сохранением исходной ошибки.

S3-совместимые хранилища

Работает с AWS S3, MinIO, DigitalOcean Spaces, Яндекс.Облаком, VK Cloud и другими.

Установка

pip install neos3files

С зависимостями для разработки:

pip install neos3files[dev]

С зависимостями для тестирования:

Быстрый старт

import asyncio
from neos3files import S3Config, S3Manager

async def main():
    # Конфигурация подключения
    config = S3Config(
        endpoint_url="https://s3.example.com",
        bucket="my-bucket",
        access_key="your-access-key",
        secret_key="your-secret-key",
    )

    async with S3Manager(config) as manager:
        # Загрузка файла
        info = await manager.upload_file(
            "local_file.txt", "remote/file.txt"
        )
        print(f"Загружен: {info.key}, {info.size_mb} МБ")

        # Проверка существования
        exists = await manager.exists("remote/file.txt")

        # Скачивание
        await manager.download_file("remote/file.txt", "downloaded.txt")

        # Копирование и перемещение
        await manager.copy_file("remote/file.txt", "remote/backup.txt")
        await manager.move_file("remote/file.txt", "archive/file.txt")

        # Удаление
        await manager.delete_file("remote/backup.txt")

asyncio.run(main())

Архитектура

Библиотека состоит из двух основных слоёв, разделяющих низкоуровневые операции и высокоуровневые абстракции:

КомпонентУровеньНазначение
S3Client Низкий Асинхронный клиент S3 с пулом соединений. Предоставляет прямой доступ к API S3: head_object, put_object, delete_object, copy_object, list_objects, multipart upload.
S3Manager Высокий Менеджер с автоматической многокомпонентной загрузкой, отслеживанием прогресса, пакетными операциями и статистикой использования.

S3Config

Конфигурация подключения к S3-хранилищу. Содержит все параметры, необходимые для установки соединения.

endpoint_urlstr—URL S3-совместимого хранилища
bucketstr—Имя бакета
access_keystr—Ключ доступа (Access Key ID)
secret_keystr—Секретный ключ (Secret Access Key)
regionOptional[str]NoneРегион AWS (по умолчанию us-east-1)
verify_sslboolTrueПроверка SSL-сертификатов

Исключения

Все исключения наследуются от S3Error. Каждое исключение имеет атрибут message с описанием и опциональный original_error с исходным исключением.

S3ErrorExceptionБазовое исключение для всех ошибок S3
S3ConnectionErrorS3ErrorОшибка подключения к S3-сервису
S3UploadErrorS3ErrorОшибка загрузки файла (содержит атрибут key)
S3DownloadErrorS3ErrorОшибка скачивания файла (содержит атрибут key)
S3FileNotFoundErrorS3ErrorФайл не найден в бакете
S3BucketNotFoundErrorS3ErrorБакет не найден
S3PermissionErrorS3ErrorНедостаточно прав для операции

from neos3files import (
    S3FileNotFoundError, S3PermissionError,
    S3ConnectionError, S3Error,
)

try:
    info = await manager.get_file_info("important.txt")
except S3FileNotFoundError:

NeoS3Files

A high-level async Python library for working with S3-compatible storage

Python 3.7+ MIT License v2.0.0 AsyncIO

Fully Async API

All I/O operations are non-blocking, built on asyncio, aioboto3, and aiofiles.

Automatic Multipart Uploads

Large files are automatically split into parts and uploaded concurrently with controlled parallelism.

Progress Tracking

Callbacks to monitor upload progress with completed parts count and percentage information.

Connection Pooling

Automatic HTTP connection reuse for maximum throughput and reduced latency.

Typed Exceptions

Fine-grained exception hierarchy for all S3 error types, preserving the original error context.

S3-Compatible Storage

Works with AWS S3, MinIO, DigitalOcean Spaces, Yandex Cloud, VK Cloud, and more.

Quick Start

import asyncio
from neos3files import S3Config, S3Manager

async def main():
    # Connection configuration
    config = S3Config(
        endpoint_url="https://s3.example.com",
        bucket="my-bucket",
        access_key="your-access-key",
        secret_key="your-secret-key",
    )

    async with S3Manager(config) as manager:
        # Upload a file
        info = await manager.upload_file(
            "local_file.txt", "remote/file.txt"
        )
        print(f"Uploaded: {info.key}, {info.size_mb} MB")

        # Check existence
        exists = await manager.exists("remote/file.txt")

        # Download
        await manager.download_file("remote/file.txt", "downloaded.txt")

        # Copy and move
        await manager.copy_file("remote/file.txt", "remote/backup.txt")
        await manager.move_file("remote/file.txt", "archive/file.txt")

        # Delete
        await manager.delete_file("remote/backup.txt")

asyncio.run(main())

Architecture

The library consists of two main layers, separating low-level operations from high-level abstractions:

ComponentLevelPurpose
S3Client Low Async S3 client with connection pooling. Provides direct access to the S3 API: head_object, put_object, delete_object, copy_object, list_objects, multipart upload.
S3Manager High Manager with automatic multipart uploads, progress tracking, batch operations, and usage statistics.

S3DownloadErrorS3ErrorFile download failed (has key attribute)
S3FileNotFoundErrorS3ErrorFile not found in bucket
S3BucketNotFoundErrorS3ErrorBucket not found
S3PermissionErrorS3ErrorInsufficient permissions
S3ConfigurationErrorS3ErrorInvalid connection configuration

pytest

# Code formatting
black neos3files tests
